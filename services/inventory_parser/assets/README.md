# Inventory Parser Assets

Provide template crops from a representative Blue Archive inventory screenshot.

Required files:
- `list_header.png` — the "List" header tab (right panel).
- `item_cannot_use.png` — the "Item cannot be used here" banner.
- `digits/0.png` .. `digits/9.png` — quantity digit templates.

Tip: crop tightly around the target region and keep consistent sizes for digits.
